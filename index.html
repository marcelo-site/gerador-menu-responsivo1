Gerador de Código para seu Menu Mobile

Algumas considerações:

Para criar link para o watsapp é nescessário esse link: https://api.whatsapp.com/send?phone=55
e completar o link com seu numero, comecando pelo DDD.

Para criar um link para o Email depois do mailto: vem seu seuemail exemplo:
mailto:usuario@gmail.com.br.

Para criar link para o telegram esse é o link: https://t.me/nome-do-usuario basta
editar nome-do-usuario para seu nome de usuário no telegram.
